# ربات فروش اشتراک تلگرامی

ربات تلگرامی برای فروش خودکار اشتراک کانال‌های پرایوت با تایید دستی رسید توسط ادمین.

## 📋 امکانات

- پنل کاربری با ۵ گزینه: ۳ محصول + نرخ ارز + پشتیبانی
- هر محصول: توضیحات → انتخاب پلن (۱/۶/۱۲ ماهه) → صفحه پرداخت (قیمت + ۳ آدرس ولت)
- ارسال رسید (عکس) توسط کاربر → ارسال خودکار به گروه ادمین با دکمه تایید/رد
- تایید ادمین → ساخت خودکار لینک دعوت **یک‌بارمصرف** کانال و ارسال به کاربر
- رد ادمین → پیام اطلاع‌رسانی به کاربر
- نرخ ارز لحظه‌ای: دلار، طلا (BrsApi.ir) و ۱۰ ارز دیجیتال برتر (CoinGecko)

## ⚙️ مراحل راه‌اندازی

### ۱. ساخت ربات و گرفتن توکن
1. به [@BotFather](https://t.me/BotFather) پیام دهید → `/newbot` → نام و یوزرنیم بدهید
2. توکن دریافتی را در env به نام `BOT_TOKEN` قرار دهید

### ۲. گرفتن آیدی عددی گروه ادمین
1. یک گروه بسازید، ربات را به آن اضافه و **ادمین** کنید
2. یک پیام در گروه بفرستید
3. همان پیام را به ربات [@JsonDumpBot](https://t.me/JsonDumpBot) فوروارد کنید
4. عددی شبیه `-1001234567890` می‌گیرید → آن را در `ADMIN_GROUP_ID` بگذارید

> ⚠️ توجه: آیدی گروه باید با علامت منفی (`-`) باشد.

### ۳. گرفتن آیدی عددی کانال‌های پرایوت
برای هر کانال:
1. ربات را به کانال اضافه و **ادمین** کنید (با دسترسی «Invite Users via Link» فعال)
2. یک پیام در کانال بفرستید، آن را به همان [@JsonDumpBot](https://t.me/JsonDumpBot) فوروارد کنید
3. عدد دریافتی را در `CHANNEL_1_ID` / `CHANNEL_2_ID` / `CHANNEL_3_ID` بگذارید

فعلاً (تا گرفتن آیدی‌های واقعی) هر سه روی یک مقدار placeholder یکسان تنظیم شده‌اند.

### ۴. گرفتن کلید رایگان BrsApi (برای نرخ دلار/طلا)
1. به [brsapi.ir](https://brsapi.ir) بروید و ثبت‌نام رایگان کنید
2. کلید را در `BRSAPI_KEY` بگذارید
3. اگر این مقدار خالی بماند، ربات فقط بخش دلار/طلا را نمایش نمی‌دهد ولی کریپتو (CoinGecko) بدون نیاز به کلید کار می‌کند

### ۵. دیپلوی روی Render

**روش پیشنهادی: Blueprint**
1. این پوشه را در یک ریپوی گیت‌هاب (GitHub) آپلود کنید
2. در داشبورد Render: New → Blueprint → ریپو را انتخاب کنید (فایل `render.yaml` خودکار شناسایی می‌شود)
3. متغیرهای env که `sync: false` دارند (`BOT_TOKEN`, `ADMIN_GROUP_ID`, و غیره) را در پنل Render دستی وارد کنید
4. Deploy بزنید

Render به صورت خودکار:
- یک دیتابیس Postgres رایگان می‌سازد و `DATABASE_URL` را وصل می‌کند
- متغیر `RENDER_EXTERNAL_URL` را ست می‌کند که ربات از آن برای وبهوک استفاده می‌کند

> 💡 توجه به محدودیت Render Free: سرویس بعد از ۱۵ دقیقه بی‌فعالیتی Sleep می‌رود و
> اولین پیام بعد از آن کمی (چند ثانیه تا حدود یک دقیقه) طول می‌کشد تا بیدار شود.
> برای رفع کامل این محدودیت در آینده، یا یک سرویس Ping خارجی (مثل UptimeRobot، رایگان)
> هر ۱۰ دقیقه به آدرس ربات درخواست بزند، یا به پلن Starter ($7/ماه) ارتقا دهید.

### ۶. تست لوکال (اختیاری)
```bash
pip install -r requirements.txt
cp .env.example .env
# مقادیر .env را پر کنید، WEBHOOK_BASE_URL را خالی بگذارید تا با Polling اجرا شود
python bot.py
```

## 🗂️ ساختار پروژه

```
.
├── bot.py                  # نقطه شروع، اتصال هندلرها، اجرای webhook
├── config.py               # همه تنظیمات (env-driven)
├── db.py                   # مدل‌های دیتابیس (User, Order)
├── keyboards.py            # کیبوردهای شیشه‌ای
├── handlers/
│   ├── start.py            # /start و منوی اصلی
│   ├── products.py         # نمایش محصول / پلن‌ها / صفحه پرداخت
│   ├── receipt.py          # دریافت عکس رسید و ارسال به گروه ادمین
│   ├── admin_actions.py    # تایید/رد توسط ادمین + ساخت لینک دعوت
│   └── exchange_rate.py    # نمایش نرخ ارز
├── services/
│   └── exchange_rate.py    # فراخوانی BrsApi و CoinGecko
├── requirements.txt
├── render.yaml              # تنظیمات Render Blueprint
└── .env.example
```

## 🔒 نکات امنیتی پیاده‌سازی شده

- هر سفارش فقط یک‌بار قابل تایید/رد است (کلیک تکراری روی دکمه نادیده گرفته می‌شود)
- فقط ادمین‌های همان گروه اجازه‌ی تایید/رد دارند (چک با `get_chat_member`)
- لینک دعوت کانال با `member_limit=1` ساخته می‌شود، یعنی فقط یک نفر می‌تواند با آن عضو شود

## ✏️ تغییرات بعدی که ممکن است نیاز شود

- قیمت/توضیحات هر محصول: در `config.py` → دیکشنری `PRODUCTS` و `PLANS`
- پیام رد رسید: `config.py` → `RECEIPT_REJECTED_MESSAGE`
- اگر خواستید قیمت‌ها برای هر محصول متفاوت باشد (الان مشترک بین سه محصول است)،
  باید ساختار `PLANS` را به ازای هر محصول جدا تعریف کنید — به من بگو تا این تغییر را هم اضافه کنم
